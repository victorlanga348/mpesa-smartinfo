# First User Experience

2. Vai para mapa ou login/auth.
3. Se nao houver sessao, fluxo do bottom sheet pode criar cliente automaticamente.
4. Mapa mostra agentes reais ou mocks.
5. Utilizador selecciona agente e inicia pedido.
