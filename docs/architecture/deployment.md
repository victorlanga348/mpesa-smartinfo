# Deployment

## Backend

Scripts:

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run prisma:migrate`
- `npm run seed`

Variaveis:

- `DATABASE_URL`
- `JWT_SECRET`
- `PORT`
- `HOST`
- `CLIENT_URL`

## Frontend

Scripts:

- `npm run dev`
- `npm run dev:lan`
- `npm run build`
- `npm run start`

Variaveis:

- `NEXT_PUBLIC_API_URL`
- `NEXT_PUBLIC_SOCKET_URL`
